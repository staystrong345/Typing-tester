

##  Features
*   **Randomized Sentences:** Pulls from a pre-defined list of technical and general sentences.
*   **Precision Timing:** Uses `time.time()` for millisecond-accurate duration tracking.
*   **Clean CLI:** Simple terminal interface that is easy to read.
